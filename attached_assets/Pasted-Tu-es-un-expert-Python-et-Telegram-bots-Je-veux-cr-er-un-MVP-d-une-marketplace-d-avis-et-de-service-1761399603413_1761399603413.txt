Tu es un expert Python et Telegram bots. Je veux créer un MVP d'une marketplace d’avis et de services en ligne avec 2 bots Telegram + 1 dashboard web admin.

CONTEXTE :
- MVP pour quelques client test 
- Budget zéro, hébergé sur Replit
- Je ne sais pas coder
- Besoin d'anonymat (pas de données perso stockées)

ARCHITECTURE MVP :

2 BOTS TELEGRAM :
1. Bot Client  - Pour l'entreprise qui commande
2. Bot Worker - Pour les exécutants (microworkers rémunérés a la tache)

1 DASHBOARD WEB ADMIN :
- Interface Flask simple (pas Telegram)
- Accès protégé par mot de passe
- Pour gérer commandes et workers facilement, et communiquer facilement les informations aux workers, inspiré du site microworkers, mais en plus sécurisé, et pas libre d’accès. 

STACK TECHNIQUE :
- Python 3.10
- python-telegram-bot (async)
- Flask (dashboard web)
- SQLite (base de données)
- Tout doit tourner sur Replit (single process)

BOT WORKER - FONCTIONNALITÉS MVP :

Message de bienvenue :
```
💰 Gagnez de l'argent en ligne

✓ Tâches simples (3-5 min)
✓ 0,15-3$ par tâche
✓ Paiement crypto

[🇫🇷 Français] [🇬🇧 English]
```

Dashboard worker :
```
👤 Profil : WRK-XXXXX
⭐ Niveau : Bronze
💰 Solde : 0.00 USDT

[💼 Tâches disponibles (3)]
[💳 Mes gains]
```

Liste tâches disponibles :
```
🟢 Tâche #TSK-XXXX
Type : Avis Google
Rémunération : 5 USDT
Durée : ~X min

✅ TEXTE FOURNI (copier-coller)

[Voir détails]
```

Exécution tâche :
1. Worker clique "Accepter"
2. Reçoit : - Instructions
   - Lien Google Maps
   - Texte exact à copier-coller
   - Note à mettre (ex: 5/5)
3. Worker publie et envoie :
   - Screenshot
   - Lien de son avis publié
4. Message : "Preuve reçue ! En attente de validation."
